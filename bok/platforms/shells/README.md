## Contributing

Have you encountered a shell-specific gotcha or best practice? Please document it here!

### How to Contribute

1. Create a new markdown file describing the issue/pattern
2. Use the template from `windows-git-bash-path-conversion.md`
3. Include:
   - Summary
   - Problem description
   - Detection methods
   - Solutions with pros/cons
   - Recommendations
   - Examples
   - Prevention checklist

4. Add link to this README
5. Submit via `.deia/submissions/pending/`
